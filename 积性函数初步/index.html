数论分块

数论分块可以在 O(N)O(\sqrt{N})O(N​)​ 的时间复杂度求出以下式子的值：

∑i=1n⌊ni⌋\sum_{i=1}^{n}\lfloor{\frac{n}{i}}\rfloor i=1∑n​⌊in​⌋

通过打表发现，有许多 ⌊ni⌋\lfloor{\frac{n}{i}}\rfloor⌊in​⌋​​ 的值是相同的且呈块状分布，设一段相同的值起始位置为 lll​​ ，则其终止位置为

$ \lfloor{\frac{n}{\lfloor{\frac{n}{i}}\rfloor } }\rfloor $

证明：

令 p=⌊nl⌋p=\lfloor{\frac{n}{l}}\rfloorp=⌊ln​⌋​，则 n=lp+cn=lp+cn=lp+c​

若存在 Δ∈NΔ \in NΔ∈N​，使得 ⌊nl+Δ⌋=p\lfloor{\frac{n}{l+Δ}}\rfloor=p⌊l+Δn​⌋=p​，则 n=(l+Δ)p+c′n=(l+Δ)p+c'n=(l+Δ)p+c′​

∴lp+c=(l+Δ)p+c′\therefore lp+c=(l+Δ)p+c'∴lp+c=(l+Δ)p+c′​

lp+c=(l+Δ)p+c′lp+c=lp+Δp+c′c=Δp+c′Δp=c−c′Δ=cp−c′p\begin{aligned} lp+c&=(l+Δ)p+c'\\ lp+c&=lp+Δp+c'\\ c&=Δp+c'\\ Δp&=c-c'\\ Δ&=\frac{c}{p}-\frac{c'}{p} \end{aligned} lp+clp+ccΔpΔ​=(l+Δ)p+c′=lp+Δp+c′=Δp+c′=c−c′=pc​−pc′​​

显然，Δmax=⌊cp⌋Δ_{max}=\lfloor{\frac{c}{p}} \rfloorΔmax​=⌊pc​⌋

∴r=l+⌊cp⌋=⌊plp+cp⌋=⌊n−pl+plp⌋=⌊n⌊ni⌋⌋\begin{aligned} \therefore r&=l+\lfloor{\frac{c}{p}} \rfloor\\ &=\lfloor{\frac{pl}{p}}+\frac{c}{p}\rfloor\\ &=\lfloor{\frac{n-pl+pl}{p}}\rfloor\\ &= \lfloor{\frac{n}{\lfloor{\frac{n}{i}}\rfloor } }\rfloor \end{aligned} ∴r​=l+⌊pc​⌋=⌊ppl​+pc​⌋=⌊pn−pl+pl​⌋=⌊⌊in​⌋n​⌋​

所以每个块的区间为：[l,⌊n⌊ni⌋⌋][l, \lfloor{\frac{n}{\lfloor{\frac{n}{i}}\rfloor } }\rfloor ][l,⌊⌊in​⌋n​⌋]​可以一直不断地跳区间即可

狄利克雷卷积

对于两个数论函数 f(x)f(x)f(x) 与 g(x)g(x)g(x)，定义其狄利克雷卷积的结果 h(x)h(x)h(x) 为：

h(x)=∑d∣nf(x)g(nd)h(x)=\sum_{d|n}f(x)g(\frac{n}{d}) h(x)=d∣n∑​f(x)g(dn​)

简记为：h=f∗gh=f*gh=f∗g

显然，狄利克雷卷积满足一下性质：

交换律 ：f∗g=g∗ff*g=g*ff∗g=g∗f

结合律：(f∗g)∗h=f∗(g∗h)(f*g)*h=f*(g*h)(f∗g)∗h=f∗(g∗h)

分配律 ： (f+g)∗h=f∗h+g∗h(f+g)*h=f*h+g*h(f+g)∗h=f∗h+g∗h

狄利克雷卷积运算的单位元：

ε\varepsilonε​​ 函数是狄利克雷卷积运算的单位元，即对于任意的数论函数 fff​，都有 f∗ε=ff*\varepsilon=ff∗ε=f

通过简单手算可知，ε(n)=[n=1]\varepsilon(n)=[n=1]ε(n)=[n=1]​
